
## Basic Print Statement:

```
print("Hello World")

```
Output:

```
Hello World
```

## Printing Variables:

```
x = 10
print(x)

```
Output:

```
10
```

## Printing text + variables:

```
name = "Roger"
print("My name is", name)

```
Output:

```
My name is Roger
```
